## ME304: BattleBot Project

ME304 공학설계 2조
배틀로봇 프로젝트

### 1. Hardware

* **MCU**: Arduino Uno
* **Sensor**: MPU-6050 (6-axis Gyro & Accelerometer)
* **Motor**: DC motor
* **Structure**: 3D printing

### 2. Software

* **Algorithm**: PID 제어


### 3. Guid

1. Basic source code `Balancing_Robot_v1.ino` 
2. Install libraries
